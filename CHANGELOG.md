## [0.1.0] - 2026-03-10

### Added
- Firefox MV3 extension scaffold for Space Manager.
- Toolbar popup with `No Space (default)` and Firefox container-backed spaces.
- Per-window space switching with tab hide/show behavior.
- Best-effort memory reduction via discarding hidden tabs.
- Last-active tab restore per space and window.
- Space-aware tab group isolation and restoration.
- Toolbar icon background that reflects the active space color.
